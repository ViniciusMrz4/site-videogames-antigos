Relatório do Projeto

Este site apresenta quatro videogames antigos e três jogos representativos de cada console. Ele foi construído com HTML, CSS e um pequeno script JavaScript para melhorar a navegação móvel.

Arquivos do projeto

  • index.html – página inicial com introdução, cards dos consoles e navegação principal.
  • ps1.html, snes.html, genesis.html, gameboy.html – páginas de consoles, cada uma com três jogos, imagens e links para vídeos.
  • style.css – define cores, tipografia, layout em grid, responsividade e estilos para cards, navegação e botões.
  • main.js – controla o botão de menu no celular para abrir e fechar o menu de navegação.
  • relatorio.html – documento que descreve a estrutura do site, as decisões de layout e a adaptação para diferentes telas.

Interligação e navegação

O menu principal aparece em todas as páginas. Cada console tem um link direto na página inicial e as páginas de console possuem link de retorno para o início. Isso garante navegação clara entre os conteúdos.

Tags HTML utilizadas

Foram usadas tags semânticas como <header>, <nav>, <main>, <section>, <article>, <figure>, <figcaption> e <footer>. Também há uso de <h1> a <h3>, <p>, <a>, <img> e <button>.

Decisões de leiaute

O site usa um layout em grade para os cards na página inicial e um layout de lista com cards nas páginas de console. As páginas mantêm um estilo consistente com cabeçalho fixo e área principal clara.

Adaptação para desktop e celular

O CSS inclui breakpoints para ajustar o menu e os cards. Em telas largas, o menu é exibido horizontalmente e os cards aparecem em várias colunas. Em telas pequenas, o menu se torna oculto por padrão e pode ser aberto pelo botão de menu; os cards são empilhados em uma coluna para leitura mais fácil.

Conclusão

O site demonstra conhecimentos de HTML semântico, CSS de layout responsivo e navegação entre páginas, além de incorporar imagens e links de vídeo para enriquecer o conteúdo.
